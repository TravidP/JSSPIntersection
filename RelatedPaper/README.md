# Related Papers for JSSPIntersection

This folder collects public PDFs and Zotero-import material for the priority-aware JSSP/AIM paper draft.

## Zotero Status

The requested `zotero` plugin was not available in this Codex session, and no Zotero plugin candidate was available to install. Because of that, the papers were **not directly imported into Zotero**.

To import manually:

1. Open Zotero.
2. Create or select a collection named `JSSPIntersection`.
3. Use `File > Import...`.
4. Select `zotero_import.bib`.
5. Add the PDFs in this folder as attachments, or drag the PDFs into the collection.

## Files

- `zotero_import.bib`: BibTeX entries copied from the verified reference list used in the LaTeX paper workspace.
- `zotero_import_with_pdf_paths.bib`: same BibTeX entries, with local `file` fields for PDFs available in this folder.
- `download_manifest.md`: PDF download/source notes.
- `*.pdf`: publicly available PDFs or a local copy from `Doc/`.
